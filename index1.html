🆕 V29 Yenilikləri
📝
Quyu Qeyd sistemi
Hər quyuya admin tərəfindən Firebase-də saxlanılan qeyd əlavə edildi. Qeyd dərhal bütün istifadəçilərə göstərilir.
🌐
Vikipediya İxracı
Hər quyu üçün {{Infobox}} şablonlu Vikipediya vikimətn avtomatik yaradılır. Qeyd daxil edilmişsə ayrıca «Qeydlər» bölməsi də əlavə olunur. Bir klikklə kopyalanır.
🚫
Balans dəyəri çıxarıldı
Subartezian və artezian quyularının detalında, redaktə formasında, yeni qurğu əlavəsində, XLSX/CSV ixracında balans dəyəri (₼) sahəsi görüntüdən çıxarıldı. DB-də mövcud olaraq qalır.
⏱️
Rele məlumatları — həmişə görünür
Subartezian quyuların detalında Zaman Relesi bölməsi həmişə açılır. XLSX_RELE statik data ilə Firebase ekstra məlumatlar birləşdirilir. Admin redaktə düyməsi birbaşa rele bölməsindən əlçatan.
📋
Digər tab strukturu yeniləndi
STQ / Anbarlar / TSTQ üçün yeni tab sistemi — Quyular səhifəsi kimi. Klik edəndə slide panel açılır, sürüşmə yoxdur. Hər tab ayrıca panel göstərir.
